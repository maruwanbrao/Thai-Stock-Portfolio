#วันที่หุ้นจำนวน (฿)หมายเหตุ
🔔 ราคาเป้าหมาย + แจ้งเตือน Line
แจ้งเตือนอัตโนมัติผ่าน LINE Notify ทุก 15 นาที
⚙️ ตั้งค่า LINE Notify (ฟรี ครั้งเดียว)
1. ไปที่ notify-bot.line.me → เข้าสู่ระบบ → "สร้าง token" → ตั้งชื่อ → เลือกห้องแชท
2. Apps Script → ⚙️ Project Settings → Script properties → LINE_NOTIFY_TOKEN = token
3. กด Deploy ใหม่ → กด "ทดสอบ Line" ด้านบน → รอ LINE เข้า ✅
#หุ้นเงื่อนไขราคาเป้าราคาปัจจุบันสถานะหมายเหตุ
₿ ติดตามคริปโต (Bitkub)
อัพเดทราคาจาก Bitkub API อัตโนมัติทุก 15 นาที
วิธีเพิ่มคริปโต: ใส่ชื่อ symbol Bitkub (เช่น BTC, ETH, XRP, DOGE) แล้วกด + เพิ่ม
ระบบจะดึงราคาจาก api.bitkub.com ฟรี ไม่ต้อง API Key
Symbolราคา (฿)เปลี่ยน 24h (฿)% 24hปริมาณอัพเดท
🤖 AI วิเคราะห์หุ้น (Claude + Web Search)
วิเคราะห์จากข้อมูลพอร์ตจริง + ข่าวล่าสุด ผ่าน Apps Script
⚙️ ต้องตั้งค่า ANTHROPIC_API_KEY ก่อน (ครั้งเดียว)
1. ไปที่ console.anthropic.com → Create key
2. Apps Script → ⚙️ Project Settings → Script properties → ANTHROPIC_API_KEY = sk-ant-...
3. Deploy ใหม่ → กด "วิเคราะห์ทั้งหมด"
