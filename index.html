#วันที่หุ้นจำนวน (฿)หมายเหตุ
AI วิเคราะห์หุ้น (Claude + Web Search)
วิเคราะห์จากข้อมูลพอร์ตจริง + ข่าวสารล่าสุด ผ่าน Apps Script
⚙️ ต้องตั้งค่า Anthropic API Key ก่อนใช้งาน (ครั้งเดียว)
1. ไปที่ console.anthropic.com → สร้าง API Key
2. เปิด Apps Script → ⚙️ Project Settings → Script properties
3. กด + Add property → Key: ANTHROPIC_API_KEY → Value: sk-ant-...
4. กด Save → Deploy ใหม่ → กลับมากด "วิเคราะห์ทั้งหมด"
