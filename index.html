🔔 แจ้งเตือนราคา (Telegram)
แจ้งเตือนอัตโนมัติผ่าน Telegram Bot ทุก 15 นาที — ฟรี
⚙️ ตั้งค่า Telegram Bot (ฟรี ครั้งเดียว ~3 นาที)
ขั้นที่ 1 — สร้าง Bot: เปิด Telegram → ค้นหา @BotFather → พิมพ์ /newbot → ตั้งชื่อ → คัดลอก Token
ขั้นที่ 2 — หา Chat ID: เปิด bot ที่สร้าง → กด Start → เปิดเบราว์เซอร์ไปที่:
  https://api.telegram.org/bot<TOKEN>/getUpdates → หา "chat":{"id": คัดลอก id
ขั้นที่ 3 — ใส่ใน Apps Script: ⚙️ Project Settings → Script properties → เพิ่ม 2 รายการ:
  TELEGRAM_BOT_TOKEN = Token จากขั้นที่ 1
  TELEGRAM_CHAT_ID = Chat ID จากขั้นที่ 2
ขั้นที่ 4: Deploy ใหม่ → กด ✈️ ทดสอบ Telegram → รอข้อความใน Telegram ✅
#หุ้นเงื่อนไขราคาเป้าราคาปัจจุบันสถานะหมายเหตุ
